# Eassential Addons Pro Widgets Marker

## Description

This WordPress plugin is designed to enhance the user experience of Essential Addons for Elementor by visually marking the pro widgets in the Elementor dashboard. It provides a clear distinction between free and pro widgets for users.

## Features

- Adds a visual indicator to pro widgets in the Elementor dashboard.
- Improves user awareness of pro features available in Essential Addons for Elementor.

## Installation

1. Upload the `ea-pro-elements-marker` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.

## Usage

No additional configuration is needed. The plugin will automatically mark pro widgets in the Elementor dashboard.
